Detailed Description

Artificial Neural Network Class

This is a class to represent a time-discrete recurrent neural network. You can subclass this class to obtain classes for special network configurations (such as VRN, PSN, CPG ...). It should be sufficient to include this file into your project. Under normal circumstances you don't need to know anything about the underlying neuron and synapses classes.
