Detailed Description

Time Discrete Artificial Neuron Class

This class represents a single neuron in a time-discrete artificial neural net. It is described by its current output, activity, bias and input values as well as by the list of incoming and outgoing synapses and the transfer function to use for this neuron.

ATTENTION: All methods assume that there are no two synapses with the same postsynaptic and presynaptic neurons.
