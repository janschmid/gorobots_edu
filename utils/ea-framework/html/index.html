Evolutionary algorithm framework.

Introduction

This framework implements the definition of a chromosome class as well as the evolutionary algorithm class. The first one is developed in order to ease the implementation of the second one and abstract some of the needed algorithms. The framework uses the gnuplot libraries to output a graph of the fitness function live while evolving so that the user can have a feedback of how the algorithm is performing. Also, for implementing the option of a multithreaded evolution the http://threadpool.sourceforge.net/ from boost.
